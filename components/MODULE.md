# components

## What this is

Android-native Kotlin/Compose component library for Yalla UI components that
need Android ownership, Android presentation behavior, or Android resource and
design adapters.

The module may reuse platform-agnostic Compose components from
`uz.yalla.sdk:components` when that avoids duplicating shared UI primitives.

## What this is not

- Not a Kotlin Multiplatform module.
- Not a Swift/iOS package.
- Not a place for shared domain or networking code.
